Installation Instructions

Requirements:
* Node.js >= v5.3.0
* NPM >= v3.3.12

```
> $ git clone https://github.com/practicalprogramming-io/node-twitter-map.git
> $ cd node-twitter-map
> $ cp config.json.tmpl config.json
# Edit config.json to add your Twitter API developer credentials
> $ npm install
> $ node server.js
```
